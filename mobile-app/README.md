# Project H Mobile App

This folder provides a native mobile client starter (Expo React Native) aligned with the campaign command platform.

## Run

1. Install dependencies:
   npm install
2. Start Expo:
   npm run start
3. Open on Android/iOS emulator or Expo Go.

## Current Scope

- Mobile command shell
- Election status snapshot integration
- Ready for adding auth, task execution, messaging, and geolocation flows
